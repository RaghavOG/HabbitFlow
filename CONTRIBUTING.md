# Contributing to HabitFlow

Thanks for taking the time to contribute to HabitFlow.

## How to contribute

1. Fork the repo and create a new branch from `main`.
2. Make your changes with clear, focused commits.
3. Run the local checks:
   - `npm run lint`
   - `npx tsc --noEmit`
4. Open a Pull Request using the PR template.

## Development

To run locally:
- `npm install`
- `npm run dev`

Then open the app in your browser.

## Notes

- Please keep PRs small and easy to review.
- Be mindful of cross-platform behavior (web + iOS/Desktop packaging).
